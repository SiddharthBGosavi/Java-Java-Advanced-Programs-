# ☕ Java Programming Practice Repository

This repository contains a collection of Java programs created while learning core Java concepts, object-oriented programming, control structures, user input handling, AWT/Swing GUI programming, and basic problem-solving techniques.

## 📚 Topics Covered

- Java Basics
- Variables and Data Types
- Arithmetic Operators
- Conditional Statements
- Command Line Arguments
- User Input using Scanner
- Matrix Operations
- GUI Development using Swing
- Graphics using AWT
- Event-Based Components
- Basic Calculator Application

---

## 📂 Programs Included

### 1. Hello World
A simple Java program that prints "Hello, World!" on the console.

### 2. Arithmetic Operators
Demonstrates:
- Addition
- Subtraction
- Multiplication
- Division
- Pre-increment Operator
- Post-increment Operator

### 3. Conditional Statements
Examples of:
- if statement
- if-else statement
- Nested conditions
- Finding largest number among multiple values

### 4. Matrix Addition
Program to:
- Accept two matrices from the user
- Add them
- Display the resultant matrix

### 5. Calculator
A console-based calculator supporting:
- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

Uses:
- Scanner Class
- Switch Statement
- Input Validation

### 6. Command Line Arguments
Demonstrates how to pass and access arguments from the command line.

### 7. CheckBox Example (Swing)
Simple GUI application using:
- JFrame
- JCheckBox

### 8. AWT Graphics Example
Demonstrates graphics drawing using:
- JPanel
- Graphics Class

Draws:
- Rectangle
- Oval
- Line
- Text

## 🚀 How to Run

### Compile

```bash
javac FileName.java
```

### Execute

```bash
java FileName
```

Example:

```bash
javac Calculator.java
java Calculator
```
